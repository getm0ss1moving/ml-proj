# Project Documentation Index

This directory contains local project-maintenance notes for the optimized LeRobot checkout.

- [Structure](./STRUCTURE.md) describes the intended repository layout.
- [Maintenance](./MAINTENANCE.md) records cleanup rules and disk-space practices.
- [Bimanual Training](./BIMANUAL_TRAINING.md) documents the two-arm, three-camera training layout.

The upstream LeRobot user and API documentation remains under `docs/source/`. Agent-facing guidance remains in the repository root as `AGENTS.md` and `AGENT_GUIDE.md` so tools can discover it without extra configuration.
